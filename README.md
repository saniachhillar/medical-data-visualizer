# Medical Data Visualizer

A Python data visualization project that analyzes medical examination data using Pandas, NumPy, Matplotlib, and Seaborn. This project was completed as part of the **FreeCodeCamp Data Analysis with Python Certification**.

## Overview

The project processes medical examination data to create meaningful visualizations and analyze relationships between cardiovascular disease, body measurements, lifestyle factors, and key health indicators using Python and industry-standard data visualization libraries.

## Features

- Calculate Body Mass Index (BMI) and identify overweight individuals
- Normalize cholesterol and glucose levels
- Create categorical bar plots for medical and lifestyle variables
- Generate correlation heatmaps for medical indicators
- Analyze relationships between cardiovascular disease and health factors
- Perform data cleaning and outlier removal
- Visualize health-related patterns using statistical graphics

## Running the Project

Install the required dependencies:

```bash
pip install pandas numpy matplotlib seaborn
```

Run the project:

```bash
python3 main.py
```

Run the unit tests:

```bash
python3 test_module.py
```
